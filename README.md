# rollup集成ts
1.安装typescript：npm i typescript
2.安装ts转换器：rollup-plugin-typescript2
3.配置typescript文件：tsconfig.json

# jest测试

2.安装@babel/core 和 @babel/preset-env: npm install @babel/core @babel/preset-env --save-dev
